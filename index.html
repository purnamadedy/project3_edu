APA SAJA YANG AKAN KAMU PELAJARI DI EDUTECH ?

Phase 0

HTML, CSS & Bootstrap

Kamu akan memahami bagaimana HTML bekerja menggunakan library bootstrap sama seperti bagaimana web kami di buat.

Phase 1

Javascript & Jquery

Implementasi programming client side - front end yang akan membantu kamu membangun aplikasi berbasis web pada saat sudah memahami back end dalam PHP.

phase2

PHP Basic

Di PHP Basic, kamu akan mempelajari native PHP seperti syntax, variabels, data type, function, array hingga session dan juga studi kasus.Jadwal kamu dari hari senin sampai jumat adalah - 2 jam teori. - 3 jam tugas individu (didampingi expert). - 3 jam tugas kelompok phase 2 (didampingi expert).

Phase 3

PHP Intermediate

Dalam PHP Intermediate, kamu akan mempelajari MVC Laravel - PHP, VueJs, SCRUM methodology dan juga studi kasus.Jadwal kamu dari hari senin sampai jumat adalah - 2 jam teori. - 3 jam tugas individu (didampingi expert). - 3 jam tugas kelompok phase 2 (didampingi expert).

Phase 4

PHP Advance

Di Phase 4 ini kamu sudah membangun portfolio dari kasus-kasus yang ada di dunia ini seperti membangun web app ojek online, marketplace, scrapper.
